Hexo主题编写1--入门

  •   由于自己系统是macos系统，所有以下描述，以macos系统为准。
  •   第一，你必须知道，如何更改hexo主题【博客根目录‘_config.yml’文件中‘theme’项】
      第二，安装了yo模块【用于生成主题基础文件】
  1. 安装yo的名命令：
  2.   npm install -g yo
  3.   说明一下 -g 参数,这个参数是全局安装的意思，如果不加上-g，yo模块就会安装在，你执行命令所在的目录下的‘node_modules’文件夹下。我的hexo是局部安装，所以我把yo设置成局部安装.
  •   开始动手制作：在hexo博客目录下，themes文件夹下，新建文件夹，取名就随意了，如”ForE”，然后在这个文件夹下，执行以下命令：
  1.   yo heox-theme
  •   接着它会提供一些主题的基本文件类型【模板引擎：EJS/Jade/Swig等，CSS 预处理：SASS/LESS/Stylus/css等】，安装提示，填写主题名称【最好和文件夹名一样】，选择好模板引擎和css预处理
  •   生产完成后，在这个文件夹（取名‘ForE’,主题名和文件夹名一样）下，就会生成主题的基本文件，现在这些文件，就可以直接用在博客上了，修改hexo配置theme项为‘ForE’【注意配置‘theme： ForE’,ForE前边是有空格的】
  •   修改完成后，在hexo博客文件夹下，执行命令：
  1. hexo s [hexo server的简写]
  •   浏览器访问：localhost:4000【hexo默认本地运行地址】，就可以访问到自己的博客。
  •   接下来，你就可以一边修改主题的文件，一边通过浏览器查看修改的情况。命令运行窗口，会在有错误的时候提醒你【有时候也可以用来测试不理解的变量：console.log()】
  •   ps：如果有任何问题【关于本文】，请联系我。
